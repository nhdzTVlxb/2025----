# 2025年睿抗智海算法调优
作者参加2025年度-睿抗的智海算法调优赛道，今年的题是语音情绪识别，刚开始给出的题目预告任务并不明显，作者在一些以前的情绪识别比赛上找了一些作为预备，比较幸运的是在我预备的方案中就找到了这题目的出处；
赛制只有三天时间来作答，作者觉得这个赛制的限制，并且没有给出一个完整的框架（我有点记不得了，反正我用的是别的开源的），反而脱离了深度调优的目的，更多的是去根据比赛或者实战经验去找方法；
作者在kaggle找到了这个题目的源头，并数据集的出处就是来自kaggle竞赛，只是只选取了部分少量数据集；
可能官方也有在低样本的背景上来调优这个目的，但是赛制并没有限制不允许扩充数据集，导致作者觉得这个反而是一个不好的现象；
上面只是作者的个人想法，不予实际评价~
作者仅仅在官方给出的数据集基础上就达到了F1:0.81（官方的排名依据）的分数，后面用了XGboost更是达到了0.95左右，但是由于平台的问题，导致我们最后的交不上去，这里开源我的做法；
作者后来也试过了，仅仅是扩充数据集，一个很基础的0.71-0.74分数的模型就能够达到0.9+的分数，所以作者觉得竞赛这方面，有时候选择大于努力；但是比赛期间作者没有去使用，去避免这个纠葛，反而是这些走漏洞的人拿到了很高的排名；
